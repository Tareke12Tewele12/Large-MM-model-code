# LLM

**LLM** is a command-line tool and Python library designed to interact with modern Large Language Models from providers such as **OpenAI**, **Anthropic Claude**, **Google Gemini**, **Meta Llama**, and many others.

It supports both **cloud APIs** and **locally hosted models**, making it suitable for experimentation, automation, and integration into development workflows.


## Key Features

LLM provides a simple interface for working with language models directly from your terminal.

- Run prompts directly from the command line  
- Save prompts and responses in **SQLite** for reproducibility  
- Generate and store **vector embeddings**  
- Extract structured information from **text and images**  
- Enable models to execute external **tools**  
- Extend functionality using a flexible **plugin system**
